# Персонажи (characters)

В этой папке хранятся конфигурации персонажей. Создавая `.yaml` файлы, вы можете переключаться между разными AI-персонажами, голосами и моделями.

## Как это работает

Проект загружает `conf.yaml` как базовую конфигурацию. При переключении персонажа в интерфейсе применяются настройки из выбранного `.yaml` в папке `characters`.

**Указывайте только то, что хотите изменить** — остальное наследуется из базовой конфигурации.

## Создание персонажа

1. Создайте `.yaml` файл (например, `my_character.yaml`)
2. Заполните поля по образцу из существующих конфигов

### Минимальный пример

```yaml
character_config:
  conf_name: 'Мой персонаж'
  conf_uid: 'my_char_001'   # уникальный ID
  character_name: 'Алексей'
  persona_prompt: |
    Ты добрый и весёлый собеседник.
```

### Основные поля

| Поле | Описание |
|------|----------|
| `conf_name` | Название в списке выбора |
| `conf_uid` | Уникальный ID (обязательно!) |
| `character_name` | Имя AI в диалоге |
| `persona_prompt` | Описание личности и стиля общения |
| `avatar` | Файл аватара (в папке `avatars/`) |

В конфиге персонажа можно переопределить ASR, TTS, LLM и другие настройки.
